# MiMo Reports

Every completed MiMo task must create a detailed report in this folder.

Naming format:

```text
YYYY-MM-DD-issue-XXX-short-title.md
```

Example:

```text
2026-07-08-issue-002-project-skeleton.md
```

Use template:

```text
agent_workspaces/mimo/templates/MIMO_REPORT_TEMPLATE.md
```

A task is not considered complete until a report exists.

## Required report sections

```text
Issue
Agent
Date / Time
Task Summary
Context Read
Files Changed
Files Created
Files Deleted
What Was Completed
What Was Not Completed
Commands Run
Test / Check Results
Safety Check
Risks
Open Questions
Next Steps
Handoff / Review Needed
Final Status
```

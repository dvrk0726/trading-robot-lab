# Ideas Library

## Назначение

Здесь хранятся сырые идеи, наблюдения, гипотезы и направления для будущих стратегий.

Идея — это еще не стратегия. Идея может быть неполной и непроверенной.

## Когда писать сюда

Записывать сюда, если есть:

- новая гипотеза;
- наблюдение по рынку;
- идея из старого HFT-робота;
- идея из книги/статьи/материала;
- подозрение на неэффективность;
- вариант улучшения уже существующей стратегии;
- новая связка инструментов;
- новый тип сигнала.

## Требование

Каждая идея должна создаваться отдельным `.md` файлом по шаблону:

```text
IDEA-YYYYMMDD-001-short-name.md
```

Использовать шаблон:

```text
IDEA_TEMPLATE.md
```

## Статусы идеи

```text
raw                     сырая идея
needs_clarification     нужно уточнить
ready_for_strategy_spec можно превращать в стратегию
rejected                отклонено
merged_into_strategy    перенесено в стратегию
```

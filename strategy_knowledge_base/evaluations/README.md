# Strategy Evaluations

## Назначение

Здесь хранятся результаты проверки стратегий: бэктесты, replay, стресс-тесты, анализ комиссий, задержек и рисков.

Оценка стратегии должна быть отдельным `.md` файлом.

## Формат файлов

```text
EVAL-YYYYMMDD-001-strategy-name.md
```

Использовать шаблон:

```text
EVALUATION_TEMPLATE.md
```

## Главный принцип

Стратегия не считается перспективной, пока она не выдержала проверку на:

- комиссии;
- проскальзывание;
- задержки;
- качество данных;
- разные рыночные периоды;
- реалистичность исполнения;
- риск крупных убытков.

## Возможные решения

```text
continue_research
ready_for_replay
ready_for_paper
needs_rework
reject
```

# Strategy Packages — Examples

Example Strategy Packages.

## Purpose

Contains reference packages that demonstrate the Strategy Package standard. Used for testing package loading, validation, and signal parity.

## Structure

Each package follows:

```text
strategy_package/
  manifest.yaml
  params.yaml
  risk_limits.yaml
  instruments.yaml
  trade_agent
  validation_report.json
  test_vectors/
  approval.json
  package.hash
```
